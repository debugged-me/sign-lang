# Filipino Sign Language Learning and Recognition System

A web-based **Filipino Sign Language (FSL)** learning, recognition, and practice system with **AI-powered interaction**, built using **CodeIgniter**, **MariaDB**, and **Python-based sign prediction**.

This project is designed to help users learn and practice Filipino Sign Language through an interactive platform that combines lesson management, sign dictionaries, webcam-based recognition, quizzes, and progress tracking. It aims to provide an accessible and engaging way to study FSL while demonstrating how artificial intelligence can be integrated into a web-based educational system.

## Features

- Filipino Sign Language sign dictionary
- Lesson and category management
- Webcam-based sign recognition
- AI-assisted sign prediction
- Practice and quiz modules
- User progress tracking
- Admin management for signs, lessons, and datasets
- Recognition logs and translation history

## Purpose

The main goal of this project is to create a practical and interactive **Filipino Sign Language learning platform** that supports both education and AI-assisted recognition. It focuses on beginner-friendly sign categories such as the alphabet, numbers, greetings, and common words, making it suitable as both a learning tool and a capstone or research-based system.
